# get-hypesquad

A simple command-line tool to change or remove your Discord HypeSquad house.

## Features

- Change your HypeSquad house to Bravery, Brilliance, or Balance
- Remove yourself from HypeSquad houses
- Simple command-line interface with parameter validation
- Displays username from the provided token
